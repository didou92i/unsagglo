

## Ajouter du relief au logo

Appliquer des effets CSS sur l'image du logo dans `src/pages/home/index.tsx` pour creer un effet de profondeur :

- `drop-shadow` via Tailwind (`drop-shadow-2xl`) pour une ombre portee douce
- Un filtre CSS personnalise avec plusieurs couches de `drop-shadow` en `#009de6` pour un halo lumineux subtil
- Optionnel : leger effet de scale au hover pour renforcer la sensation 3D

Fichier concerne : `src/pages/home/index.tsx` (classe CSS sur la balise `<img>`)
